\begin{abstract}
    Este informe detalla el diseño e implementación de estrategias de control digital avanzado para un sistema \emph{ball-and-beam}. A partir de un modelo linealizado y discretizado a \qty{50}{\Hz}, se desarrolló un observador de Luenberger que permitió la estimación precisa de estados y la atenuación de ruido en las velocidades. Se implementó un control por realimentación de estados, mejorado posteriormente con una ganancia de \emph{feed-forward} para el seguimiento de referencias. Finalmente, se incorporó acción integral para compensar el error en estado estacionario producido por la fricción. Los ensayos experimentales validaron el diseño, logrando estabilizar el sistema con un tiempo de establecimiento inferior a \qty{2}{\s} y un sobrepico menor al \qty{15}{\percent}, cumpliendo con los objetivos planteados por la cátedra.
\end{abstract}
